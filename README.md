# Sim Utility Hub Pro 🏎️

Um hub centralizado e moderno para gerir todos os softwares essenciais de Sim Racing.

## ✨ Funcionalidades
- **Grid & List View:** Escolha a visualização que preferir.
- **Favoritos:** Mantenha as suas ferramentas principais no topo.
- **Auto-Sync:** Links atualizados via Gist.
- **Smart Path:** Deteção automática da pasta de Downloads.

## 🛠️ Tecnologias
- Electron.js
- HTML5 / CSS3 (Glassmorphism)
- JavaScript (Vanilla)
