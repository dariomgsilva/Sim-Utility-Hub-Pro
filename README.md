## 📖 The Story Behind the Project
As a Sim Racing pilot, I faced a recurring problem: every time I formatted my PC or reinstalled Windows to gain performance, I wasted hours jumping from site to site to download the same essential tools (SimHub, iOverlay, CrewChief, Drivers, etc.).

I created **Sim Utility Hub Pro** to be my "survival kit." It is a centralized application that gathers all essential software in one place with always-up-to-date links, ensuring that moving to a new OS takes minutes, not hours.

## ✨ Key Features
- **🚀 Optimized for Reinstalls:** Have all your tools ready after formatting your PC.
- **🔄 Auto-Sync Server:** Download links are fed via Gist (always the latest version).
- **📂 Smart Path Detection:** Automatically detects your Windows Downloads folder to avoid errors.
- **⭐ Favorites System:** Pin your most-used software to the top.
- **📱 Dual View Modes:** Choose between Grid mode (Logos) or List mode (Compact).
- **🔍 Instant Filter:** Find exactly what you need in seconds.

## 🚀 How to Install
1. Go to the [Releases](https://github.com/dariomgsilva/Sim-Utility-Hub-Pro/releases) section.
2. Download the `Sim Utility Hub Pro Setup 1.0.0.exe` file.
3. Run the installer.
4. (Optional) If you prefer not to install, download the **Portable (.zip)** version.
